# Final Lecture Folder

This folder holds the lecture material that should stay visible and easy to find.

## Main files

- `THzTDS_Lecture_Fit_Study_final.ipynb`
  - final lecture notebook used for the fit/study lecture workflow
- `lecture_thz_tds_fit_study_notes_revised.tex`
  - final revised lecture-note source
- `lecture_thz_tds_fit_study_notes_revised.pdf`
  - compiled lecture-note PDF

## Generated lecture sources

The `generated/` subfolder contains the auto-generated lecture notebook and source files used by the lecture asset builder:

- `generated/THzTDS_Lecture_Fit_Study.ipynb`
- `generated/lecture_thz_tds_fit_study_notes.tex`
- `generated/lecture_thz_tds_fit_study_slides.tex`

## Figures and study outputs

The lecture notebook and the final notes both read figures and saved outputs from:

- `../lecture_build/`

That build folder stays outside this directory so the lecture artifacts and the reusable build outputs remain clearly separated.
