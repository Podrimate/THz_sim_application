# Slim Lecture Build Snapshot

This folder is a deliberately reduced lecture-build snapshot.

It contains the measured-fit outputs, selected study figures, and carrier-density maps that are used by the final lecture notes in `../final_lecture/`.

It does not contain the full historical lecture build from the larger working repository. That full build was too large to keep in a clean branch intended for long-term GitHub use.

If you rerun the lecture notebook or the lecture asset builder, this folder can be regenerated and expanded locally as needed.
